The Night

The Night is a personal project that was born in April of 2016. It is currently under development.

This game gathers some ideas from Silent Hill PT game and Paranormal Activity movie, what makes it into a survival horror game.

It is being made in Unreal Engine, using C++ as programming language. Meshes are being modelled and texturized with 3D Studio Max, 3D Coat and Adobe Photoshop.

SOULS

SOULS is a mystery-horror game where you will have to look for proofs in order to discover what happened in an old abandoned house. You will have to interact with some objects and think about which will be your next step.

It was developed for the Games subject of the Computer Graphics, Games and Virtual Reality masters degree with another three colleagues.

This game was made in Unity, using C# as programming language. Due to the short period of time given to develop it, the most of the meshes were downloaded from different websites. Some other were modelled in 3d Studio Max. Texturing was made with Adobe Photoshop.

Pirates of the Caribbean

Pirates of the Caribbean was a project made for a masters degree subject.

The aim of the game is to destroy the enemy defences, which are located in an island in middle of nowhere, with the well-known ship The Black Pearl.

Despite their horrible aim, the tower and the fort won’t stop shooting, so be careful...they sometimes hit the mark.

This game has been made with Unity, using C# as programming language.

The ship and other elements, such as the fort or the tower, were modelled with 3D Studio Max and texturized with Adobe Photoshop.

Mixed Reality

This was my master's degree final project. The aim of this research is to provide some tools which WEARHAP project could investigate with.

In this case, the tool provided is a virtual scene where the user can interact with several objets using his hands through a sensor device. This device is Leap Motion tracker.

Leap Motion supports hand and finger motions as input, analogous to a mouse, but requires no hand contact or touching.
